 Aplicatie pentru Raport cu interfata grafica, versiunea 1.0.0.
 
 Aplicatia este impartita in 2 parti:
 
•Partea de sus unde se afla butoanele si informatiile.
    •In stanga vor aparea butoane cu ID-urile tehnicienilor dupa ce le-a fost pornit shift-ul si informatii despre codurile la care lucreaza (locatia unde a fost asignat si de cat timp a fost asignat).
    •Pentru a selecta un tehnician pentru a il asigna la coduri, se apasa mai intai pe butonul cu ID-ul tehnicianului.
    •Pentru a porni sau inchide turele tehnicienilor se bifeaza Start shift sau End shift, se scrie ID-ul tehnicianului in campul text si se apasa tasta enter.
    •Pentru a asigna un tehnician la un cod, mai intai se selecteaza tehnicianul apoi se selecteaza bifa de Asignat si se completeaza campul Cod Locatie si optional campul Comentariu, dupa care se apasa tasta Enter sau se apasa pe butonul Scrie.
    •Pentru a edita direct ce scrie in raportul unui tehnician, se selecteaza tehnicianul, se scriu modificarile in partea de jos si se apasa pe butonul Update.
    •Butonul Genereaza raport va genera raportul pentru toti tehnicienii activi si il va salva in folder-ul aplicatiei.
 
•Partea de jos unde se vor afisa rapoartele tehnicienilor. Aceasta este editabil.
